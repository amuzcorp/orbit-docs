# CMS-Orbit Introduction

CMS-Orbit is a powerful content management system based on Laravel and Orchid. The system has the following key features:

## Key Features

- **Orchid-based Admin Panel**: Intuitive and extensible admin interface
- **Multilingual Support**: Built-in multilingual support structure
- **Theme System**: Flexible theme system based on Inertia.js
- **SEO Optimization**: Metadata management and SEO optimization tools
- **Extensible Architecture**: Modular architecture for easy extension

## System Requirements

- PHP 8.3 or higher
- Laravel 11.x
- Composer
- Node.js 18.x or higher
- MySQL 8.0 or higher or PostgreSQL 13 or higher
