# 멀티사이트

멀티사이트 기능을 사용하면 하나의 애플리케이션에서 여러 웹사이트를 관리할 수 있습니다.

## 주요 기능

- **사이트 생성**: 새로운 사이트를 쉽게 생성하고 설정
- **사이트 관리**: 사이트 설정, 사용자, 콘텐츠 관리
- **사이트 설정**: 도메인, SSL, 캐시 등 설정
- **보안**: 사이트별 보안 설정 및 접근 제어
- **유지보수**: 백업, 업데이트, 모니터링
- **문제 해결**: 일반적인 문제 해결 및 디버깅

## 시작하기

멀티사이트 기능을 사용하기 위해서는 다음 단계를 따르세요:

1. [사이트 생성](/multi-site/site-creation) - 새로운 사이트 생성 방법
2. [사이트 관리](/multi-site/site-management) - 사이트 관리 방법
3. [사이트 설정](/multi-site/site-settings) - 사이트 설정 방법
4. [보안](/multi-site/security) - 보안 설정 및 관리
5. [유지보수](/multi-site/maintenance) - 백업 및 업데이트
6. [문제 해결](/multi-site/troubleshooting) - 일반적인 문제 해결

## 모범 사례

1. 각 사이트에 대한 명확한 네이밍 규칙 사용
2. 정기적인 백업 수행

4. 성능 모니터링
5. 문제 발생 시 로그 확인
